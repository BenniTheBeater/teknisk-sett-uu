GitHub
Hvordan virker GitHub?
Git virker med det som kalles for "kommandoer" eller "commands".
Det vil si at det er delt opp på en spesiell måte.
Hva er en commit?
En såkalt "commit" er et slags snapshot av hvordan ting så ut på ett hvis punkt i tid.
Det er veldig nyttig hvis du gjør feil og må tre tilbake.
