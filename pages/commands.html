List of Commands
Add
Syntax:
$ 
git 
add
 [path]
Function: Stages the specified folder/files.
Commit
Syntax:
$ 
git 
commit
 -m
 [message]
Function: Makes a commit of the staged files with the specified message/name.
Remote
Syntax:
$ 
git 
remote 
add 
[name] 
[url] 
Function: Links a remote repository to a local one.
Push
Syntax:
$ 
git 
push
 [origin]
 [branch]
Function: Pushes the staged files to the specified remote repository.
Pull
Syntax:
$ 
git 
pull
 [origin]
 [branch]
Function: Pulls changes from remote repository into the local repository.
A heads up
This page doesn't go into detail on all commands, and is therefore not a reliable source of information,
but rather an example of how some commands might be used.

It is therefore recommended that you take a look at official documentation over at the Git Docs.
Here they go into detail on how all commands work, and how to use them.
