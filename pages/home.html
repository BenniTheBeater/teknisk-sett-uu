Universell Utforming
Av Julian, Benjamin, Henrok, Morten og Daniel.
Hva er universell utforming?
Universell utforming er et prinsipp som handler om å lage produkter
og tjenester som er tilgjengelige for alle, uavhengig av funksjonsevne.
Det handler om å skape løsninger som er inkluderende og tilpasset ulike behov.
På denne siden ønsker vi å vise hvordan UU kan implementeres.
Lesbar skrifttype
En lesbar skrifttype er viktig for å sikre at innholdet er lett å lese for alle.
Det anbefales å bruke skrifttyper som Arial, Verdana eller Helvetica.
Disse skrifttypene er enkle og klare, noe som gjør dem
lettere å lese for personer med synsproblemer.
God skriftstørrelse
En god skriftstørrelse er avgjørende for lesbarheten.
Det anbefales å bruke en minimum skriftstørrelse på 16px.
Dette sikrer at teksten er lett å lese, selv på små skjermer.
I tillegg bør det være tilstrekkelig avstand mellom linjene for å forbedre lesbarheten.
Tilrettelegg for skjermleser
Tekst blir automatisk lest opp av skjermleseren.
Men for å sikre at andre bilder og elementer også blir lest opp,
Bør du bruke alt-tekster og ARIA-attributter.
Who Uses Git
Git is the backbone of version control for software developers worldwide.
It's the go-to tool for individual programmers, small teams, and large organizations.
From open-source projects to tech giants, Git is a universal language of collaboration and code management.
If you're involved in software, you're likely using Git.
Gode kontraster
God kontrast fokuserer på at det er tydelig visuell forskjell mellom elementer
Det brukes for å gjøre innhold lestbart, tilgjengelig og visuelt interessant.
Det er viktig med tanke på fargeblinde og svaksynet mennesker.
Tydelig og strukturert innhold
Det er viktg med beskrivende oversktifter og lenketekster, ikke skriv "Klikk her", men f.eks "Les mer om saken her!"
Unngå å ha vegger med tekst, del opp teksten med underoverskrifter, punktlister og kortere avsnitt.
Bruk riktige HTML-strukturelementer, h1 til h6, lister med ul/ol, og avsnitt med p
Responsivt design
Innholdet på nettsiden skal fungere på mobil, nettbrett og store skjermer uten problemer
Brukerne skal kunne zoome til hva enn de vil uten at noe blir ødelagt
