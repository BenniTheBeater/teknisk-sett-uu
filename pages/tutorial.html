Tutorial
Project Setup
First of all, you'll need to create a git repository.
You can do this with the command
$ 
git 
init
 .
Committing
Before committing, you'll first have to stage your files using the command
$ 
git 
add
 [path]
 .
Once you have staged your wanted files, you can make a commit using the following command:
$ 
git 
commit
 -m
 [message]
 .
