How does Git work?
Git virker med det som kalles for "kommandoer" eller "commands".
Det vil si at det er delt opp på en spesiell måte.
What is a commit?
A commit is like a snapshot of how your code looked at a point in time.
It is an essential part of Git, and is definitely one of the most used Git commands.
The syntax is pretty simple:
$ 
git 
commit
 -m
 [message]
Where
commit
 is the only required part.
